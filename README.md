# woo-commerce-top-3-user-products
Adds a shortcode [my_top_products] you can add anywhere to display the users top 3 products including processing and purchased orders.

# Built-in to add a new My Account tab called 'My Products' where users top 3 products will display. Optional to remove this part and just add the shortcode you can use anywhere on your website. 

## To use, after adding this code to functions.php file, flush permalinks under setting->permalinks. (Go to this tab, do not change anything and hit save). This assumes your permalink structure is /my-example/ type. If it is /my_example/ or any other type please adjust the code for this, or this will display a 404 error.
